== Description ==

Provides a simple and very lite social login system for your Wordpress without any server overloads !

= Features =

*	Integration with the most popular social platforms Facebook, Twitter and Google
*	Step by step documentation for all social login platforms
*	Fancy plugin UI and good user Experience
*	Easy to upload and customize your social  login platforms icons
*	Support getting the email of Twitter users
*	Ability to enable/disable any platform
*	Very lite plugin on your server so we wrote it very optimized code
*	Well database structure design for big websites
*	Option to write your customized CSS code
*	Auto integration with all of your login forms

== Installation ==

= From your WordPress dashboard =
1. Upload the plugin files to the `/wp-content/plugins/smio-social-login` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `Simpla Social` Name screen to configure the plugin.

= Once Activated =
1. Go to `Simpla Social under `Tools` tab in your dashboard sidemneu.
2. Enable the desired platforms and you can find a documentation link beside each platform option.
